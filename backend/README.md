# SmartVigilant 🛡️

**The world's first dual-purpose AI security platform** — protecting both your **digital life** and **physical safety** with autonomous, self-improving intelligence.

Built January 2026.

### Features
- Real-time malware & agentic AI attack detection
- Deepfake and phishing protection
- Live home surveillance with AI motion/face/behavior analysis
- Natural disaster early warnings
- Emergency panic button with multi-channel alerts
- AI-vs-AI autonomous threat neutralization
- Privacy-first federated learning
- Cross-platform: iOS, Android, Desktop

### Tech Stack
- **Frontend**: Flutter (Dart)
- **Backend**: FastAPI (Python)
- **AI Engine**: PyTorch, TensorFlow, ONNX, YOLO11, InsightFace
- **Database**: PostgreSQL + Alembic
- **Deployment**: Docker, docker-compose

### Quick Start
```bash
# Clone and run
git clone https://github.com/yourname/smartvigilant.git
cd smartvigilant

# Start full stack
docker-compose up --build

# Or run separately
# Backend
cd backend && uvicorn app.main:app --reload

# Frontend
cd frontend && flutter run
